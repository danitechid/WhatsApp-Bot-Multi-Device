## Installation
### Clone project
```bash
$ git clone https://github.com/danitechid/WhatsApp-Bot-Multi-Device.git
```
### Path to folder project
```bash
$ cd WhatsApp-Bot-Multi-Device
```
### Install dependencies
```bash
$ npm i
```
### If error, use yarn
```bash
$ yarn install
```
### Run project
```bash
$ npm run start
```
